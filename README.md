## Project Overview
This repository showcases an interactive sales performance dashboard built entirely in Microsoft Excel, featuring pivot tables and dynamic charts.  I built this modern Dashboard that has multiple interactive-style visual tiles. It is a McDonald's Sales Dashboard for South America (2022). Section 1 contains the Title, section 2, the KPI tiles,section 3 has the sales trend chart, 4 the customer satisfaction radar chart and 5 is the Sales by Country map. This dashboard provides a comprehensive, visually appealing overview of McDonald's 2022 sales performance in South America, combining KPIs, trend analysis, customer satisfaction, and geographic insights. It is well-structured for executive decision-making and quick interpretation.
